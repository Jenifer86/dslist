Para iniciar na pasta dlist comandos:
1 - git init
2 - git add .
3 - git commit -m "Project created"
4 - git branch -M main -> alterar para branch main
5 - No git pegar os comandos no SSH -> git remote add origin git@github.com:Jenifer86/dslist.git  (É feito isso para associar o repositório local com o repositório remoto 
6 - git push -u origin main -> para salvar no github
7 - SSH chave criada com email mencionado e s 123456789
8 - Se ocorrer algum problema com a chave ssh -> https://support.github.com/
. para acessar o diretório das chaves digitar - "cd ~/.ssh/" e apertar Enter
. depois escrever - "ls" - para listar as chaves
. após isso digitar - "cat id_rsa.pub" OU "more id_rsa.pub" OU "vi id_rsa.pub e para sair do ESC:wq" -> comando para mostrar a chave ssh
